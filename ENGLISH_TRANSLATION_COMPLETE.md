## ğŸŒ Next Steps for i18n

If you want to implement full internationalization (i18n) with multiple languages:

### 1. Install next-intl
```bash
npm install next-intl
```

### 2. Create Translation Files
```
/messages
  /en.json
  /fr.json
  /ar.json
  /es.json
```

### 3. Example Translation File (`/messages/en.json`)
```json
{
  "admin": {
    "maids": {
      "title": "Maids",
      "subtitle": "{count} maids in total",
      "add": "Add",
      "edit": "Edit",
      "stats": {
        "total": "Total",
        "active": "Active",
        "featured": "Featured",
        "avgSalary": "Average Salary"
      },
      "table": {
        "name": "Name",
        "nationality": "Nationality",
        "age": "Age",
        "experience": "Experience",
        "salary": "Salary",
        "status": "Status",
        "actions": "Actions"
      },
      "form": {
        "save": "Save",
        "saving": "Saving...",
        "cancel": "Cancel"
      }
    }
  }
}
```

### 4. Use Translations in Components
```tsx
import { useTranslations } from 'next-intl';

export default function MaidsPage() {
  const t = useTranslations('admin.maids');
  
  return (
    <h1>{t('title')}</h1>
    <p>{t('subtitle', { count: stats.total })}</p>
  );
}
```
